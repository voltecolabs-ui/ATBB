# ATBB — Trading Scripts

This repository contains Python trading scripts for BTC/crypto market analysis and automated trading.

## Key files

- `bybit_auto_trader.py` — Main automated trading bot (Bybit exchange)
- `btc_market_regime.py` — Market regime detection
- `btc_news_rss.py` — Crypto news RSS parser
- `analysis_five_days.py` — 5-day price analysis
- `monitor_service.py` — Service monitoring

## Rules

- Python 3.x
- Do NOT commit `.pyc` files or `__pycache__/` — add them to `.gitignore`
- Keep `.backup` files for safety but don't modify them
- All changes are auto-synced to GitHub via background service
